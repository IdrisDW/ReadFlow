# ReadFlow

ReadFlow is a learning project focused on C#, ASP.NET Core, SQL, Angular, testing, debugging, Git, PRs, and basic cloud concepts.

## Goal

The goal is to practice real Software Engineer I skills through a small but well-structured project.

## Project idea

ReadFlow helps users manage books and reading progress.

## Tech stack

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- Angular
- xUnit
- GitHub
